<h1 align="center">Online Resume</h1>

<div align="center">

基于Umi + React 所开发的项目。

## 安装依赖项
```bash
npm install antd --save
```
## 启动开发服务器
```bash
yarn start
```



克隆到本地开发:

```bash
$ git clone https://github.com/chuan0106/resume.git
$ cd ant-design
$ yarn
$ yarn start
```

打开浏览器访问 http://localhost:8000
